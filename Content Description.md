1. C# Fundamentals & Logic (Assignments 1–5)
Type Mechanics: Exploring the functional differences between Value Types (Stack) and Reference Types (Heap), including deep-dives into memory allocation behavior.
Control Flow & Math: Implementation of mathematical solvers, including prime number ranges, binary converters, and custom calculators.
Data Structures: Advanced array manipulation, such as merging sorted arrays, frequency counting, and calculating the longest distance between equal elements.
Functional Design: Mastery of parameter passing mechanics using ref, out, and value parameters to control data state.

2. Object-Oriented Programming (OOP Assignments)
Encapsulation & Modeling: Designing robust classes (e.g., Employee systems) using properties, access modifiers, and custom constructors.
Structs & Enums: Utilizing lightweight data structures for coordinates (Point3D) and enums for state management (Seasons, Permissions, Security Levels).
Polymorphism & Abstraction: Implementing and inheriting from interfaces (IShape, ICloneable, IComparable) to create flexible and reusable software components.
Operator Overloading: Customizing standard operators to work with user-defined types (e.g., Duration and 3D Point objects).

3. Advanced C# & Algorithms (Advanced Assignment 1)
Generics: Development of a generic Range<T> class to handle type-agnostic data ranges using the IComparable interface.
Algorithm Optimization: Analysis and implementation of optimized sorting techniques, specifically focusing on improving the time complexity of the Bubble Sort algorithm.
